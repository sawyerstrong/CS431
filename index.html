Renaissance

Renaissance is a project I worked on for CS267, a Colby College Course. The piece was an interactive art display in which the audience approached a blank canvas and camera that took a photo of them. A neural style transfer was then used on the picture, and the audience was then "painted" in the style of a famous painting. I was the Lead Creative Director and focused on leading the project to the best final product. Learn more.

Using VHDL to make a Pseudo 3D Game

Not every project is completely intense and serious, and this remake of Pole Position in VHDL that Tyler Hansen and I lovingly dubbed "PDAMDSWAC" definitely fits that category. By using python scripts to convert complex sprites and designing a system in VHDL we were able to create a small game that simulated 3D movement and kept track of a score for players. On the surface, this project isn't crazy, but the value it has to me came with the challenge of working on an extremely low level to perform complex graphical tasks. Although we ran into issues like sprites that were too large for our software to run, this project was a ton of fun and a great challenge to develop! Find the Git Hub Repository Here.

Next Project

Shadow Puppet

Shadow Puppets is a 2D platformer in which the player starts in a completely dark map and must light the cave they are within to escape the cave. By lighting each lantern the player can move on to the next cave until it meets a Spider that avoids the player and must be found in order to leave. My primary task was dealing with the mechanics behind this spider, in addition to working on some other aspects of movement including the players. Working in a group of about 6 people, we were able to create a game that had a series of levels including a custom audio track and cut scenes creating a story that progresses through the levels. By using PyGame we were able to completely make this within Python, which was a challenge of its own. This project took around 3 weeks to develop from start to finish and the I am proud of the final product we have. Find the Git Hub repository here.
